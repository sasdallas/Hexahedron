liballoc allocator for Hexahedron
======================================

---------- DESCRIPTION ----------
liballoc is a well-known hobby OS memory allocator that uses the classic buddy-style allocation system.
It was used exclusively in reduceOS v1.x, and plays much nicer with I386 than toaru_alloc.
